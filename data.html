Data Dictionary

id

Type: integer

Description: Unique identifier for each record.

prompt_id

Type: string

Description: Identifier for the prompt (example: P001).

prompt

Type: string

Description: Full text of the prompt that produced the model output.

model

Type: string

Description: Model that produced the output.
Values: gpt-5, gemini-2.5-flash, claude-haiku-4-5

date

Type: string

Description: Timestamp when the response was generated or saved.
Format: YYYY-MM-DD_HH-MM-SS (example: 2025-12-15_14-45-31)

model_response

Type: string

Description: Full model output text.

error_type

Type: string

Description: Controlled vocabulary describing the type of error or hallucination.
Values:

  • misattribution — AI assigns a work/statement/action to the wrong person or organization.
  • erasure_by_omission — AI omits real, relevant scholars/artists/sources that should be mentioned.
  • adjacent_error — AI gives information close to correct but not what was asked (nearby location, related institution, associated work).
  • invented_figure — AI creates a person who never existed.
  • temporal_error — AI gives wrong dates or places events in the wrong period.
  • geographical_error — AI gives the wrong location (city/state/country/institution).
  • factual_error — AI provides incorrect factual details about real people, works, or events.

error_description

Type: string

Description: Free-text description elaborating on the error identified during annotation.

verification_source

Type: string

Description: URL or citation used to verify/refute the model output.

category

Type: string

Description: Topical category for the prompt/record.
Values:

  • black_art_artists
  • black_feminisim
  • black_lgbtq_history
  • historical_institutions_organizations
  • black_texts_authors
  • african_diaspora
